# Humanoid Robotics: A Practical Introduction

This repository contains the documentation for "Humanoid Robotics: A Practical Introduction", built with Docusaurus.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

This website is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

## Contributing

The documentation is organized into the following sections:
- `docs/intro/` - Introduction to humanoid robotics
- `docs/foundations/` - Foundational concepts
- `docs/ros2/` - ROS 2 fundamentals
- `docs/digital-twin/` - Simulation environments
- `docs/isaac/` - Isaac robotics framework
- `docs/vla/` - Vision-Language-Action systems
- `docs/capstone/` - Capstone project
- `docs/deployment/` - Deployment strategies
- `docs/appendices/` - Additional resources
